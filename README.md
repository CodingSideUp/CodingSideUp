## What I'm Building Now

**[Demand Forecasting System — Favorita](https://github.com/CodingSideUp/demand-forecasting)**

> Enterprise-grade demand forecasting engine on 125M+ real Ecuadorian grocery transactions (2013–2017).

- Segmented LightGBM pipeline (SBC classification: smooth / erratic / intermittent / lumpy)
- Two-stage intermittent model: binary classifier (AUC 0.993) x MSE magnitude regressor
- Calibrated P10/P90 quantile uncertainty bands (Q10 error: 0.5%, Q90 error: 0.1%)
- CatBoost M2 Full for holiday forecast projection through December 2017
- Interactive Streamlit dashboard with SHAP waterfall explainability + AI-generated insights
- **Result: 23.52% WAPE — 42% improvement over naive baseline**
